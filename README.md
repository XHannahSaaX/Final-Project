## Description

    I aimed to create a custom cursor using python code within Visual Code Studio.
The cursor will be a fun shape, in this case; a warm yellow star, and as the cursor moves there wil be "magic dust" falling from it.

The "magic dust" is a system of particles that rain down from the cursor and leave a trail as the cursor moves around the screen.
the particles have a slight horizontal movement and size difference so that it looks a bit more random than just falling straight down. It gives the cursor a more magical feel. The falling particles are rainbow colored and depending on where the cursor is on the screen, the colors of the falling particles will be different. The height of the cursor in particular is what determines the colors of the particals.   

When the cursor is clicked, an animation of an explosion of particles will appear. the explosion shoots warm colored particals in all directions from the click point before they fade away. The particles that are come from the click point will have some random feelings to them as they will have wiggle to them as they shoot away from the cursoe, so even if the button is clicked a lot, it doen't look clunky. This added more interest to the cursor and makes the cursor more fun to use, while not being too obnoxious or in the way.

Initially I wanted the cursor to be able to be used outside of VSC, but unfortunately I found that the only way to do that while still using python code was to download a different cursor maker. I'd then have to make a program that runs using a cursor made with that downloaded cursor maker. I wouldn't be able to implement the skills learned throughout this course like how I wanted to, so I oppted to just stick within the VSC made window. This allowed me to use pygame and particle classes which is what I wanted to experiment with within this project.
