## Description
I want to create an custom mouse cursor with interactive visuals. Custom cursors can add a lot of identity and charm to a brand or company and can be used within their websites.

## Features
- Feature 1: Raining particles
	- I'll have particles falling from where the cursor is on the screen
- Feature 2: Explosion of particles w/ mouse click
	- A small explosion or other animation when the mouse clicks 
- Feature N: Particles following cursor
	- The effects will need to follow and keep up with the cursor 

## Challenges
- I will need to learn how to get python to recognise a cursor
- I will need to learn how to ensure that the visuals will follow the cursor
- I will need to learn how to have the cursor be able to leave VSC and be functional elsewhere

## Outcomes
Ideal Outcome:
- A colorful cursor that has different colored and shaped particles raining down from it wherever on the screen the cursor may be. When clicking buttons, the cursor will perform a little animation with an explosion of particles. The cursor can be used outside of VSC.

Minimal Viable Outcome:
- A cursor that has colorful particles that rain from it and follow it wherever it may be on the screen.
